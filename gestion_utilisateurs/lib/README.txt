To add SendGrid dependency:
1. Download sendgrid-java-4.x.x.jar from https://github.com/sendgrid/sendgrid-java/releases
2. Place the JAR file in this lib directory
3. In your IDE (Eclipse/IntelliJ):
   - Right-click project > Build Path > Configure Build Path
   - Select 'Libraries' tab > Add JARs
   - Navigate to this lib directory and select the SendGrid JAR
   - Click Apply and Close
